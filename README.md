## 📋 معرفی

**گلد هاب** یک پلتفرم جامع، مدرن و هوشمند برای مدیریت کامل فروشگاه‌های طلا و جواهر است که توسط **گروه توسعه آریا** طراحی و توسعه یافته است. این سیستم با بهره‌گیری از جدیدترین فناوری‌های روز دنیا (هوش مصنوعی، قیمت زنده طلا، رابط کاربری واکنش‌گرا) تمام نیازهای یک طلافروشی مدرن را پوشش می‌دهد.

گلد هاب برخلاف نرم‌افزارهای سنتی و قدیمی بازار، یک پلتفرم **مبتنی بر وب**، **چندکاربره**، **چندشعبه‌ای** و **هوشمند** است که روی هر دستگاهی (موبایل، تبلت، لپ‌تاپ، دسکتاپ) به‌خوبی کار می‌کند.

### مزایای کلیدی گلد هاب

🏆 **تنها نرم‌افزار طلافروشی ایران با هوش مصنوعی**
- دستیار هوشمند: سوال درباره فروش، انبار، مشتریان با زبان طبیعی فارسی
- اسکن فاکتور با OCR: استخراج اطلاعات فاکتور از تصویر
- تشخیص محصول از تصویر: تخمین ویژگی‌های محصول طلا

💎 **قیمت زنده طلا**
- به‌روزرسانی خودکار هر ۳۰ دقیقه
- منبع: سایت‌های معتبر ایرانی (tgju، iranjib، taline)
- پوشش: طلای ۱۸/۲۴/۲۲/۱۴ عیار، دلار، اونس، سکه

🚀 **پلتفرم وب مدرن**
- بدون نیاز به نصب
- قابل استفاده روی هر دستگاهی
- رابط کاربری فارسی RTL با تم طلایی
- حالت روشن و تاریک

## 🎯 ماژول‌های کامل

### ۱. 🔐 احراز هویت و امنیت
- ورود امن با JWT Session
- کنترل دسترسی مبتنی بر نقش (RBAC)
- ۵ سطح دسترسی: مدیر ارشد، مدیر، مدیر شعبه، صندوق‌دار، کارمند
- پشتیبانی چند مستاجر (Multi-tenant)
- لاگ ممیزی کامل از همه فعالیت‌ها
- ماتریس دسترسی ماژول‌ها به نقش‌ها

### ۲. 📊 داشبورد مدیریتی
- شاخص‌های کلیدی عملکرد (KPI)
- نمودار روند فروش (۱۴ روز اخیر)
- توزیع عیارهای طلا
- پرفروش‌ترین محصولات
- روش‌های پرداخت
- **تیکر قیمت زنده طلا** (به‌روزرسانی هر ۵ دقیقه)
- **پنل کامل قیمت زنده طلا** با نمودار تاریخچه
- هشدارهای موجودی کم
- آخرین فروش‌ها

### ۳. 📦 مدیریت انبار
- مدیریت محصولات با بارکد و SKU
- پشتیبانی از ۵ عیار (۲۴، ۲۲، ۱۸، ۱۴، ۱۰)
- ثبت وزن، اجرت، سنگ، قیمت خرید و فروش
- مدیریت دسته‌بندی‌ها (با زیردسته)
- حرکات انبار (ورود، خروج، تعدیل، مرجوعی)
- هشدار خودکار موجودی کم
- ۱۴ دسته‌بندی آماده (نشان عروسی، انگشتر، گردنبند، دستبند، نیم‌ست، تنگسیر، پلاک، بچگی، گوشواره، زنجیر، پایه عروس و ...)

### ۴. 🛒 صندوق فروش (POS)
- فروش سریع با اسکن بارکد
- سبد خرید با محاسبه خودکار
- ۵ روش پرداخت (نقدی، کارتی، انتقال، طلا، ترکیبی)
- مدیریت تخفیف
- انتخاب مشتری
- **چاپ فاکتور** با بارکد
- کاهش خودکار موجودی
- رابط کاربری دو پنلی (محصولات + سبد)

### ۵. 👥 مدیریت مشتریان (CRM)
- پروفایل کامل مشتریان
- تاریخچه خرید‌ها
- **سیستم امتیازات وفاداری** (۴ سطح: پلاتین، طلایی، نقره‌ای، برنزی)
- یادآوری تولد‌ها
- **ارتباط با واتساپ** (یک کلیک)
- بخش‌بندی مشتریان بر اساس میزان خرید
- جستجو و فیلتر پیشرفته

### ۶. 📋 سفارشات و تعمیرات
- ثبت سفارش ساخت جواهر
- پیگیری تعمیرات
- **گردش کار ساخت** (طراحی → ساخت → پرداخت → آماده → تحویل)
- تایم‌لاین کامل مراحل با timestamp
- مهلت تحویل
- تخصیص به استاد زرگر

### ۷. 🏢 شعبات و انبارها
- مدیریت چند شعبه
- مدیریت انبارهای جداگانه
- **انتقال بین انباری** با گردش کار (pending → in_transit → received)
- شاخص عملکرد هر شعبه
- مقایسه فروش شعبات

### ۸. 💰 حسابداری
- مدیریت صندوق فروش (Cashbox)
- درآمد در برابر هزینه
- مدیریت هزینه‌ها (اجاره، حقوق، قبوض، تجهیزات، تبلیغات، سایر)
- **تسویه روزانه** صندوق
- نمودار سود و زیان
- گزارش‌های مالی

### ۹. 📈 گزارشات و تحلیل
- گزارش فروش (روزانه/هفتگی/ماهانه)
- گزارش انبار (ارزش، توزیع عیار، کهنگی)
- گزارش مشتریان (بخش‌بندی، نرخ بازگشت)
- گزارش عملکرد کارکنان
- گزارش مالی (سود و زیان)
- خروجی CSV
- چاپ مستقیم

### ۱۰. 🛠️ ابزارهای طلافروشی
- **مودیان مالیاتی و ارزش افزوده (VAT)**
  - محاسبه خودکار مالیات ۹٪ ارزش افزوده
  - محاسبه مالیات ۳٪ اجرت طلا
  - معافیت سکه و پلاک طلا
  - خروجی CSV و چاپ
- **مرجوعی کالا**
  - ثبت مرجوعی با بازگردانی موجودی
  - ثبت دلیل مرجوعی
  - گزارش مرجوعی‌ها
- **چاپ بارکد**
  - چاپ برچسب بارکد برای محصولات
  - انتخاب چند محصول
  - چاپ در فرمت A4

### ۱۱. 🤖 هوش مصنوعی (AI)
- **دستیار هوشمند**: پرسش با زبان طبیعی فارسی درباره فروش، انبار، مشتریان و مالی
- **اسکن فاکتور (OCR)**: استخراج اطلاعات فاکتور از تصویر با VLM
- **تشخیص محصول**: تحلیل تصویر محصولات طلا و تخمین ویژگی‌ها
- تاریخچه پرسش‌ها
- پرسش‌های پیشنهادی

### ۱۲. 🛍️ فروشگاه آنلاین (Marketplace)
- ویترین آنلاین محصولات
- فیلتر بر اساس دسته، عیار، قیمت، وزن
- جستجو و مرتب‌سازی
- علاقه‌مندی‌ها (Wishlist)
- سبد خرید
- چند فروشگاه (شعبه)
- امتیازدهی محصولات

### ۱۳. ⚙️ مدیریت سیستم
- مدیریت کاربران
- ماتریس نقش‌ها و دسترسی‌ها
- لاگ‌های ممیزی
- تنظیمات سیستم (نرخ مالیات، ارز، ...)

## 🚀 نصب و راه‌اندازی

### پیش‌نیازها
- Node.js ۱۸+ یا Bun
- npm یا bun

### مراحل نصب

```bash
# کلون کردن پروژه
git clone https://github.com/HojatJoshani/goldhub.git
cd goldhub

# نصب وابستگی‌ها
bun install
# یا
npm install

# تنظیم متغیرهای محیطی
cp .env.example .env

# راه‌اندازی دیتابیس
bun run db:push

# اجرای seed داده‌های اولیه
bun run scripts/seed.ts

# اجرای سرور توسعه
bun run dev
```

برنامه در آدرس `http://localhost:3000` اجرا می‌شود.

### راه‌اندازی سرویس قیمت زنده طلا (اختیاری)

```bash
cd mini-services/gold-price-updater
bun install
bun run dev
```

این سرویس هر ۳۰ دقیقه قیمت‌های زنده طلا را دریافت و در دیتابیس ذخیره می‌کند.

## 🤖 قابلیت‌های هوش مصنوعی

گلد هاب از `z-ai-web-dev-sdk` برای ارائه قابلیت‌های پیشرفته AI استفاده می‌کند:

### دستیار هوشمند (LLM)
- پاسخ به سوالات مدیر/فروشنده با زبان طبیعی فارسی
- دسترسی به داده‌های واقعی فروشگاه
- نمونه: «فروش امروز چقدر بوده؟»، «پرفروش‌ترین محصولات کدامند؟»

### اسکن فاکتور (OCR + VLM)
- استخراج اطلاعات فاکتور از تصویر
- شناسایی فروشنده، تاریخ، اقلام، مبلغ کل

### تشخیص محصول (VLM)
- تحلیل تصویر محصول طلا
- تخمین نوع، عیار، وزن، سبک
