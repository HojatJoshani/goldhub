# 🏆 گلد هاب | GoldHub

**پلتفرم جامع مدیریت فروشگاه طلا و جواهر با هوش مصنوعی**

A production-ready SaaS platform for gold and jewelry stores, built with Next.js 16, TypeScript, Tailwind CSS, shadcn/ui, Prisma, and AI capabilities.

---

## ✨ ویژگی‌ها (Features)

### 🔐 احراز هویت و امنیت
- ورود با ایمیل/رمز عبور و JWT Session
- کنترل دسترسی مبتنی بر نقش (RBAC): مدیر ارشد، مدیر، مدیر شعبه، صندوق‌دار، کارمند
- چند مستاجره (Multi-tenant)
- لاگ‌های ممیزی کامل

### 📊 داشبورد هوشمند
- KPI‌های کلیدی (فروش امروز، ۳۰ روز، ارزش انبار، سود خالص)
- نمودار روند فروش (Area Chart)
- توزیع عیارها (Pie Chart)
- پرفروش‌ترین محصولات (Bar Chart)
- روش‌های پرداخت
- **قیمت زنده طلا** (تیکر با به‌روزرسانی خودکار هر ۵ دقیقه)
- **پنل قیمت زنده طلا** با ۳ تب:
  - قیمت لحظه‌ای (طلای ۱۸/۲۴/۲۲/۱۴ عیار + دلار + اونس + سکه)
  - نمودار تاریخچه (با stats: فعلی/میانگین/بالاترین/پایین‌ترین/تغییر)
  - جزئیات منبع و آمار دیتابیس
- هشدارهای موجودی کم
- آخرین فروش‌ها

### 🏆 قیمت زنده طلا (Live Gold Prices)
- **منبع:** جستجوی وب از سایت‌های معتبر ایرانی (tgju، iranjib، taline، alanchand)
- **به‌روزرسانی خودکار:** هر ۳۰ دقیقه توسط mini-service پس‌زمینه
- **به‌روزرسانی frontend:** هر ۵ دقیقه (auto-refresh)
- **به‌روزرسانی دستی:** دکمه به‌روزرسانی در پنل
- **پوشش:** طلای ۱۸/۲۴/۲۲/۱۴ عیار، دلار آزاد، اونس جهانی، سکه
- **ذخیره:** همه قیمت‌ها در دیتابیس با timestamp برای تاریخچه
- **نمودار:** Area chart با بازه ۲۴ ساعت تا ۳۰ روز

### 📦 مدیریت انبار
- محصولات با بارکد و SKU
- مدیریت عیار (۲۴، ۲۲، ۱۸، ۱۴، ۱۰)
- وزن، اجرت، سنگ، قیمت خرید/فروش
- دسته‌بندی‌ها
- حرکات انبار (ورود، خروج، تعدیل)
- هشدار موجودی کم

### 🛒 صندوق فروش (POS)
- فروش سریع با اسکن بارکد
- سبد خرید با محاسبه خودکار
- چند روش پرداخت (نقدی، کارتی، انتقال، طلا، ترکیبی)
- تخفیف
- انتخاب مشتری
- **چاپ فاکتور** با بارکد
- کاهش خودکار موجودی

### 👥 مدیریت مشتریان (CRM)
- پروفایل مشتریان
- تاریخچه خرید
- امتیازات وفاداری
- یادآوری تولد
- **ارتباط با واتساپ** (یک کلیک)
- بخش‌بندی مشتریان

### 📋 سفارشات و تعمیرات
- سفارش ساخت جواهر
- پیگیری تعمیرات
- **گردش کار ساخت** (طراحی → ساخت → پرداخت → آماده → تحویل)
- تایم‌لاین مراحل
- مهلت تحویل

### 🏢 شعبات و انبارها
- مدیریت چند شعبه
- انبارها
- **انتقال بین انباری** با گردش کار
- شاخص عملکرد شعبات

### 💰 حسابداری
- صندوق فروش (Cashbox)
- درآمد در برابر هزینه
- مدیریت هزینه‌ها (اجاره، حقوق، قبوض، ...)
- **تسویه روزانه**
- نمودار سود و زیان

### 📈 گزارشات و تحلیل
- گزارش فروش (روزانه/هفتگی/ماهانه)
- گزارش انبار
- گزارش مشتریان
- عملکرد کارکنان
- گزارش مالی
- خروجی CSV
- چاپ

### 🤖 هوش مصنوعی
- **دستیار هوشمند**: پرسش با زبان طبیعی درباره فروش، انبار، مشتریان (LLM)
- **اسکن فاکتور (OCR)**: استخراج اطلاعات فاکتور از تصویر (VLM)
- **تشخیص محصول**: تحلیل تصویر محصولات طلا (VLM)
- تاریخچه پرسش‌ها

### 🛍️ فروشگاه آنلاین (Marketplace)
- ویترین محصولات
- فیلتر بر اساس دسته، عیار، قیمت، وزن
- جستجو و مرتب‌سازی
- علاقه‌مندی‌ها (Wishlist)
- سبد خرید
- چند فروشگاه (شعبه)

### ⚙️ مدیریت سیستم
- مدیریت کاربران
- ماتریس نقش‌ها و دسترسی‌ها
- لاگ‌های ممیزی
- تنظیمات سیستم

## 🛠️ تکنولوژی‌ها (Tech Stack)

| لایه | تکنولوژی |
|------|----------|
| فریمورک | Next.js 16 (App Router) |
| زبان | TypeScript 5 |
| استایل | Tailwind CSS 4 |
| کامپوننت | shadcn/ui (New York) |
| آیکون | Lucide React |
| نمودار | Recharts |
| دیتابیس | Prisma ORM + SQLite |
| احراز هویت | JWT Session (Custom) |
| هوش مصنوعی | z-ai-web-dev-sdk (LLM + VLM) |
| فونت | Vazirmatn (فارسی) |
| تم | طلایی/کهربایی، روشن/تاریک |

## 🚀 نصب و اجرا

### پیش‌نیازها
- Node.js 18+ یا Bun
- npm / bun

### مراحل

```bash
# کلون کردن پروژه
git clone https://github.com/hojatjoshani/goldhub.git
cd goldhub

# نصب وابستگی‌ها
bun install
# یا
npm install

# تنظیم متغیرهای محیطی
cp .env.example .env

# اجرای migration و seed دیتابیس
bun run db:push
bun run scripts/seed.ts

# اجرای سرور توسعه
bun run dev
```

برنامه در `http://localhost:3000` اجرا می‌شود.

## 🗄️ مدل‌های دیتابیس

- **Tenant** — مستاجر (فروشگاه)
- **User** — کاربر با نقش
- **Branch** — شعبه/انبار
- **Category** — دسته‌بندی محصول
- **Product** — محصول (با عیار، وزن، اجرت)
- **Supplier** — تامین‌کننده
- **Sale / SaleItem** — فروش و اقلام
- **StockMovement** — حرکت انبار
- **Transfer** — انتقال بین انباری
- **Customer** — مشتری
- **CustomOrder / OrderTimeline** — سفارش و گردش کار
- **Cashbox / CashboxTransaction** — صندوق و تراکنش
- **Expense** — هزینه
- **Invoice** — فاکتور
- **GoldPrice** — قیمت طلا
- **Alert** — هشدار
- **Notification** — اعلان
- **AuditLog** — لاگ ممیزی
- **AIQuery** — پرسش‌های هوش مصنوعی

## 🤖 قابلیت‌های هوش مصنوعی

این پروژه از `z-ai-web-dev-sdk` برای قابلیت‌های AI استفاده می‌کند:

1. **دستیار هوشمند (LLM)**: پاسخ به سوالات فروشنده/مدیر با زبان طبیعی فارسی، با دسترسی به داده‌های واقعی فروشگاه
2. **اسکن فاکتور (OCR + VLM)**: استخراج اطلاعات فاکتور از تصویر
3. **تشخیص محصول (VLM)**: تحلیل تصویر محصول طلا و تخمین ویژگی‌ها
